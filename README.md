# Planejamento das Ferramentas de Desenvolvimento

Linguagem base: O front-end está sendo desenvolvido com HTML, CSS e JavaScript, enquanto o PHP é utilizado no back-end para processar dados e gerenciar a comunicação com o banco de dados.
Bibliotecas e frameworks: Está sendo utilizado o Bootstrap, que facilita a criação de layouts modernos, responsivos e padronizados, agilizando o desenvolvimento da interface.
Integração com o back-end: A integração ocorre por meio de requisições POST, que garantem o envio seguro de dados entre as páginas e o servidor PHP.
Controle de versão: O código é versionado e compartilhado por meio do GitHub, permitindo colaboração, rastreamento de alterações e backup do projeto.
Organização do código: O desenvolvimento é feito no Visual Studio Code, com pastas organizadas por função, como: /css, /js, /fonts, /images.
